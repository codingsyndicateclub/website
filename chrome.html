About

CSS is a simple club that aims to bring coders alike together to tackle either course work problems or your own personal projects. The club runs on collaboration, and having fun programming! Club activities include website development, app development, and even game programming.

CSS is CCR approved! As an active member of CSS, you will be able to add the club as an extra-curricular CCR activity on your transcript!

CSS has also formed Sheridan's first ACM chapter! We hope to collaborate, create and host events to further our studies in programming. If you would like to join the chapter or have an idea for a club event or ideas of what we could cover in the club meet, please send us an email!

Practice

Blockly

Blockly is great if you're new to programming and want more practice through several puzzles.

Design

Moqups

Used in the designing stages of an app, Moqups provides templates and hotspots to mock the functionality of a real app.

App Development

Ratchet

Ratchet allows you to make mobile apps using HTML, CSS and javascript.

Cordova

Cordova allows the user to create apps through HTML, CSS and javascript.

JQuery Mobile

JQuery Mobile provides a cleaner and faster way to develop websites designed for the mobile experience.

Android Studio

Android Studio is an IDE used for developping apps for the android platform.

Genymotion

Genymotion is a nicer, faster android emulator.

Collaboration

CodeShare

Need to build code collaboratively? CodeShare allows real-time code building for groups.

Game Development

Unity

Unity is a free game engine used to develop both 2D and 3D games.

Unreal Engine

Unreal Engine is a free game engine used to develop both 2D and 3D games.
